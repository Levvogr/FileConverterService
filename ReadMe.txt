Преобразует данные из xml файла в json или наоборот

xml файл представляет собой иерархическую структура из биологии начиная с класса заканчивая видом, но только для класса насекомых. То есть по порядку класс, отряд, семейство, род, вид. Смотри пример ins.xml

json файл представляет набор названий видов и куда они входят, то есть их класс, отряд, семейство, род, вид. Смотри пример jsf.json
